protected

Map that enables quick searches of the beam database BoostStore.

Keys are entry numbers in the BeamData TTree, values are (start time, end time) pairs giving the range of times (in ms since the Unix epoch) recorded for the E:TOR875 device (used to determine POT values) in each entry
