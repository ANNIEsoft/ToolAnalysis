155  // Map storing data from the beam monitoring instruments for the spill
156  // of interest. Keys are device names, values are pairs where the
157  // first element is a timestamp for the measurement (in ms since the Unix
158  // epoch) and the second element is a BeamDataPoint object.

160  // ** Known device names (from the Intensity Frontier beam database) **

162  // signal indicates that the Booster is preparing to send a beam
163  // pulse to ANNIE).
164  //
165  // E:HI860, E:VI860, E:HP860, E:VP860 = beam intensity (I) or position (P)
166  // monitor reading at position 860.
167  // Horizontal (H) or vertical (V)
