00185 /*
00186 ok, crux of the matter here:
00187 We want to make cumulative plots that can show how an 'average' CCQE event
00188 differs from an 'average' CCNPi event.
00189 
00190 For plotting 'CCQE' vs 'CCPi' hitmaps on the same canvas, we need to have blue/red marker colours
00191 that distinguish the event type, so we do not need a collection of TPolyMarkers, just one. XXX
00192 
00193 If we want to compare more than just their postions, we must plot the two on separate canvases.
00194 But what else is there to plot in such a case? 
00195 Hit time doesn't make sense: the pion lifetime is so short all parents are ejected simultaneously.
00196 Hit charge likewise; features would be washed out, rather than enhanced, accumulating over many events.
00197 Charge from parentage doesn't even make sense for a CCQE event.
00198 
00199 A cumulative plot of charge from parentage for CC1PI events *would* be worthwhile;
00200 for this we just need to accumulate the appropriate polymarkers. XXX
00201 */
