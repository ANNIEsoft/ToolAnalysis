188 /*
189 ok, crux of the matter here:
190 We want to make cumulative plots that can show how an 'average' CCQE event
191 differs from an 'average' CCNPi event.
192 
193 For plotting 'CCQE' vs 'CCPi' hitmaps on the same canvas, we need to have blue/red marker colours
194 that distinguish the event type, so we do not need a collection of TPolyMarkers, just one. XXX
195 
196 If we want to compare more than just their postions, we must plot the two on separate canvases.
197 But what else is there to plot in such a case?
198 Hit time doesn't make sense: the pion lifetime is so short all parents are ejected simultaneously.
199 Hit charge likewise; features would be washed out, rather than enhanced, accumulating over many events.
200 Charge from parentage doesn't even make sense for a CCQE event.
201 
202 A cumulative plot of charge from parentage for CC1PI events *would* be worthwhile;
203 for this we just need to accumulate the appropriate polymarkers. XXX
204 */
